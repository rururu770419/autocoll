設定手順:

  1. 上記QRコードでBotを友だち追加
  2. 管理者に「友だち追加完了」を報告
  3. 管理者がLINE管理画面でUser IDを確認
  4. 左フォームの「LINE User ID」欄にUser IDを入力
  5. 通知設定を選択して保存

👤 管理者向け: User ID確認方法

  1. LINE Official Account Manager (https://manager.line.biz/) にログイン
  2. @275gmabd を選択
  3. 「分析」または「メッセージ」メニューから友だち一覧を確認
  4. 新しく追加された友だちのUser ID（Uで始まる文字列）をコピー
